# Issue 001: Product README, Counts, And Hero

## Parent

`real-life-workflows-004`

## What to build

Create the operator-facing product README for Real Life Workflows. It should
lead with the promise to search a large workflow universe, show a polished hero
asset, advertise the verified scale without overclaiming, and link to a
project-local count artifact that explains the 19,000+ upstream-flow claim.

## Acceptance criteria

- [ ] README opens with a product-like hero and the headline "Search a large workflow universe."
- [ ] README references the generated local hero image.
- [ ] README shows 19,000+ upstream flows, 171 curated workflow doors, and 17 source repositories.
- [ ] README says the 19,000+ number is a source-universe estimate, not a fully audited workflow count.
- [ ] README does not say or imply "171 repos."
- [ ] README includes a compact breakdown of the 171 curated doors without listing every door.
- [ ] README links to the project-local count artifact.

## Blocked by

None - can start immediately.
